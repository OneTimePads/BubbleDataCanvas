Scroll: Fast Zoom In/Out
Drag Empty Space / Bubble Body: Pan Canvas
Drag Center ⊕ Gizmo: Move Bubble (Drag outside or deep into another to reparent)
Drag Bubble Outer Border: Resize / Extend Bubble
Double-Click: Add Nested Bubble (sized to current zoom)
Shift + Double-Click: Add a Smaller Nested Bubble
Right-Click: Edit Bubble Text (clear it to delete)
Save/Load: Optional password encrypts the file via custom XOR-CBC
